# xenv

This is a simple collection of dot files and scripts for a simple 
personalized development environment setup on Unix-like systems.

## Features

- Zsh configuration with Oh My Zsh
- Custom aliases and functions
- Vim configuration with plugins
- Git configuration
- Tmux configuration
- Useful scripts for development tasks

## Environment

Please refer to the `dotenv.example` file for the list of environment variables
used in this setup. Copy it to `.env` and modify the values as needed.

By default, the setup taked the `.env` file


## Programming Language Support

Following language environments are supported:

- Rust 
- Python3
- Go 
- Node.js
